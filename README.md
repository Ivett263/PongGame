# Unity 2D Pong Game

Welcome to the Unity 2D Pong game! 
This simple project was created to explore Unity's capabilities in a 2D game development environment.

## Features

- **Player vs AI:** Enjoy a classic Pong experience with a player-controlled paddle and an AI-controlled opponent.
- **Music with Volume Slider:** Groove to the beats with in-game music and control the volume using a handy slider.
- **Player Name Input:** Personalize your gaming experience by entering your name in the game.

## Controls
- Player 1:
Move Up: W 
Move Down: S 

- Player 2 (AI):
AI controls the second paddle.

- Volume Slider:
Adjust the volume using the in-game volume slider.

- Player Name Input:
Enter your name using the provided input field.
